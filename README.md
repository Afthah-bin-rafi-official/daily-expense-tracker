## ✨ Features

### 🧾 Expense Management
- Add & manage daily expenses
- Categorized spending (Fuel, Food, Travel, etc.)
- Real-time total calculation
- Clean tabular view
- One-click delete

### 💰 Debt Tracker
- Track **borrowed & lent money**
- Add return dates
- Smart reminder banner
- Total debt calculation

### 📊 Analytics Dashboard
- 📈 Expenses by category
- 📉 Expenses by day
- 👥 Debts by person
- Toggle between:
  - Weekly view
  - Monthly view

### 🎨 UI Highlights
- 🌑 Dark Carbon Mint theme
- ⚡ Smooth animations
- 📱 Fully responsive
- ✨ Modern glassmorphism cards

## ⚙️ How It Works

- No backend required
- Fully offline after load

## ▶️ Run Locally

```bash
git clone https://github.com/Afthah-bin-rafi-official/daily-expense-tracker.git
cd daily-expense-tracker
```
Open index.html in your browser.
